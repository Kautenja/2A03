\section{Overview}

SN76489 is an emulation of the SN76489 sound chip from the Sega Master System (SMS) for VCV Rack. The SN76489 chip contains three pulse wave generators and a noise generator with LFSR.

SN76489 provides the key features of the SN76489 chip, namely,
\begin{itemize}
  \item \textbf{Triple pulse wave generator:} Dual pulse waves with 10-bit frequency and $50\%$ duty cycles;
  \item \textbf{Noise generator:} generate pseudo-random numbers at 4 different frequencies; and
  \item \textbf{Linear Feedback Shift Register (LFSR):} for that old-school 8-bit randomness!
\end{itemize}

\begin{enumerate}
  \item Coarse frequency control over the four channels.
  \item $V$/Octave inputs for pulse waveform generators.
  \item linear CV frequency modulation for pulse waveform generators.
  \item Attenuation control. Silences the channel as the input value increases.
  \item LFSR switch. When high, LFSR is enabled.
  \item CV LFSR gate, high at $2V$. Holds the LFSR generator as long as the input voltage is $>2V$. Inverts when LFSR switch is high
  % \item Period of randomness $\in [0, 15]$ for the noise generator. See \url{https://wiki.nesdev.com/w/index.php/APU_Noise} for approximate frequency and pitch mappings.
  \item Channel outputs, ${\approx}10V_{pp}$.
\end{enumerate}
